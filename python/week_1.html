La sentencia `if` se usa para ejecutar c√≥digo solo si una condici√≥n es verdadera. Puedes a√±adir `elif` (else if) para m√∫ltiples condiciones y `else` para un caso por defecto.

  • Indentation es CLAVE: Python usa la indentaci√≥n (espacios al inicio de la l√≠nea) para definir bloques de c√≥digo. ¬°No uses tabulaciones y espacios mezclados!

4.2 La Sentencia `for` (Bucles)

La sentencia `for` se usa para iterar (recorrer) sobre los elementos de una secuencia (como una lista o una cadena) o sobre otros objetos iterables.

for fruta in frutas:
    print(f"Me gusta la {fruta}")

# Iterar sobre una cadena

    print(letra)

# 4.2.1 La funci√≥n range()
# Genera secuencias de n√∫meros. Muy √∫til con for.

    print(i)

for i in range(2, 6): # Genera 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2): # Genera 0, 2, 4, 6, 8 (paso de 2)
    print(i)

4.2.2 `break`, `continue` y `else` en Bucles

Estas palabras clave te dan m√°s control sobre c√≥mo se comportan los bucles:

  • `break`: Sale completamente del bucle.
  • `continue`: Salta el resto del c√≥digo de la iteraci√≥n actual y pasa a la siguiente.
  • `else` en bucles: El bloque `else` se ejecuta si el bucle termina *normalmente* (es decir, no por un `break`).
# Ejemplo con break
for numero in range(1, 10):
    if numero == 5:
        print("Encontr√© el 5, ¬°saliendo del bucle!")
        break
    print(numero) # Imprime 1, 2, 3, 4

# Ejemplo con continue
for i in range(1, 6):
    if i % 2 == 0: # Si es par
        print(f"Saltando el n√∫mero par {i}")
        continue # Pasa a la siguiente iteraci√≥n
    print(f"N√∫mero impar: {i}") # Imprime 1, 3, 5

# Ejemplo con else en bucle for
for n in range(5):
    print(n)
else:
    print("El bucle termin√≥ sin un 'break'.")

# Ejemplo con else en bucle for (con break)
for n in range(5):
    if n == 3:
        print("Encontr√© el 3, rompiendo el bucle.")
        break
    print(n)
else:
    print("Este mensaje NO se imprimir√°.")
